Creating Table:

    CREATE table job (
        jobTitle VARCHAR(50),
        jobponsor VARCHAR(50),
        jobponsorEmail VARCHAR(50),
        jobAffiliation VARCHAR(50),
        jobTask VARCHAR(400),
        jobDescription VARCHAR(400),
        jobDuration VARCHAR(20)
        jobID SERIAL UNIQUE,
    );

Delete !ALL! data inside the table:
    
    TRUNCATE TABLE job;

Adding data inside the table:

    INSERT INTO job VALUES(Title, Description, StartDate,EndDate, Wage, Qualification, id)

Deleting a row (USING UNIQUE ID):

    DELETE FROM job WHERE id = 'SpecificID'

Adding Email to the Table Data:

    SELECT * FROM job;
    UPDATE job SET email=concat(email,',', 'EmailFromUser') where ID = 'IDofTheNewUser'
